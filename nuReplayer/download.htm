Main Package

The main package contains Windows binaries and FreeBASIC source code, and also the last turn of each finished seasonal championship and arena game. (11 turns total)

Version 1.09 is compatible with 1.04+ CSV files. However, it is worth mentioning that strings containing quotation marks (") did not convert correctly. Re-convert the offending game(s) if you have this problem.
