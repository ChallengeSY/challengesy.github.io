Main Packages

To make subsequent packaging easier, we have opted to ditch the Lite vs. Deluxe system. The Main Package contains Windows binaries and FreeBASIC source code, but no games.

Version 0.91 (Beta) is compatible with 0.90+ CSV files, but exceptionally old pre-History turns did not convert correctly in 0.90. Reconvert with MassTurn, if you have this problem.
