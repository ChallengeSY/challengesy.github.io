Main Package

The main package contains Windows binaries and FreeBASIC source code, and also the last turn of each finished seasonal championship match. (8 turns total)

Version 1.02 is compatible with 1.00+ CSV files. However, it is worth mentioning Horwasp planets did not convert correctly in older builds. Reconvert if you have this problem.
