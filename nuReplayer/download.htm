Main Package

The main package contains Windows binaries and FreeBASIC source code, and also the last turn of each finished seasonal championship match. (8 turns total)

Version 1.01a is compatible with 1.00+ CSV files. However, it is worth mentioning Horwasp planets did not convert correctly in older builds. Reconvert if you have this problem.

Disclaimer! Microsoft (and only Microsoft) aggressively flags the primary binary of this package as a trojan horse! The source code can be reviewed. Still, caution is advised.
