Main Packages

To make subsequent packaging easier, we have opted to ditch the Lite vs. Deluxe system. The Main Package contains Windows binaries and FreeBASIC source code, and also the last turn of each finished seasonal championship match. (6 turns total)

Version 0.94 (Beta) is compatible with 0.90+ CSV files; but exceptionally old pre-History turns did not convert correctly in 0.90, and debris disk maps did not convert correctly in pre-0.62 builds. Reconvert with MassTurn, if you have one or both problems.
