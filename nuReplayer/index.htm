Welcome to the Nu Replayer, a detailed history program for Planets Nu. Nu Replayer will be designed to permit navigation among the turns of the completed games, and analyze the statistics for each player at those times.

Available Files

Project Features

  • The ability to view the starmap and statistics of any completed Planets Nu game
  • Can go all the way back to the beginning of the game. (Assuming the game was started on/after February 3, 2011.)
  • Data that is converted is automatically exported to CSV files that may be opened in any compatible program. Also used as cache
  • Can download an entire game, including downloading and extracting the ZIP file on the spot
