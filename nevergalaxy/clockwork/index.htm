Neverball Clockwork Summary

Neverball Clockwork was a custom build of Neverball that focused on the included collectible clocks. Its foundation is now included in the core Neverball foundation.

Here is what is included in the final standalone build, Neverball Clockwork 1.6.0.4:

  • Pink-colored collectible clocks. They extend game time as shown when collected
  • The game clock now counts down correctly at ten minutes (10'00''00) and above
  • Two level sets (50 levels total) that were formerly part of Neverball Forever
  • All master commits from October 4, 2015, and earlier

Notes

  • The windows build is a minimalist build, and does not come with ZIP support, language support, the octocat ball, or any other compile-time features.
  • The 50 levels included also work in Nuncabola.
  • Clockwork Remix is a temporary set comprising of levels that do not use clocks from Clockwork Alpha and Clockwork Beta. This set is compatible with official Neverball builds starting with 1.6.0. It will no longer be updated, now that Neverball Clockwork has been integrated with the core project.
  • Compatibility notes: (Obsoleted, but kept as a reference)
    • Levels compiled in a non-clockwork build work correctly in Neverball Clockwork, and (to a lesser extent) vise versa
    • Clocks (when compiled from a clockwork build) appear in a non-clockwork build as red (or blue) coins that are meaningless, effectively making them troll coins in normal play
    • Replays made with Neverball Clockwork playback accurately in a non-clockwork build (with the increased time), despite the previous note
    • Replays made in a non-clockwork build on a level whose clocks would appear in Neverball Clockwork play correctly, aside from the time not increasing in playblack
