Instructions

Simply play around with this program and design your own unique ship, as if you were using it in a Close Encounters game. The Summary box lists ship specifications.

Abilities which are shared with another ship (pre-Advanced Construction) must also be researched before the ship can be bought. Additionally, there can be a maximum cost of 32 CP, the maximum cost that Ship Size 7 allows. Finally, designs can have a maximum of 2 abilities.

The program supports creating and importing ship codes. This makes it possible to link to specific unique ship designs.
