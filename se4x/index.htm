Welcome to the Space Empires Resourcepedia, a supplementary collection of resources designed to augment the experience of the board game and related computer games

Sub-project List

Doomsday Machine Battle Simulator

This program supports a near-through simulation of a player fleet versus a single Doomsday Machine, of any strength.

Replay Center

This program allows viewing playthroughs in a more interactive format. The oldest playthroughs have been migrated, and there is a fresh playthrough also.

Unique Ship Designer

This program allows easily designing your own Unique Ship, as if you were playing a Close Encounters expansion game.
