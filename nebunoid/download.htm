System Notes

Nebunoid has been tested on Windows XP, and also 7-11. 512 MB RAM is recommended, though it is quite possible to coast with less RAM by passing in a parameter.

-k {MB} tells Nebunoid to stop pre-loading new soundtrack once the RAM reaches below the {MB} threshold. -s tells the program to skip loading the soundtrack entirely.
