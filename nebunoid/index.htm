Features

Difficulty Slider system

The difficulty is set on a decimal system, allowing for a lot of flexibility. The system also comes equipped with a Handicap system, allowing local hotseat matches to play on different settings.

Password/Level select system

A majority of the included levels (but not all) come with a password. A level select system is also included, virtually eliminating the need to remember passwords. These can be used to start a more casual playthrough at a later start point than the first level.

Per-level variation system

Each level can be customized with their own variations. Some variations are obvious, while others are not as obvious.

Level Editor

The included level editor can be used to build more levels. Although the controls are not that simple, it is still fairly flexible. Block brushes are on a per-level basis, and connections can be established to build multi-hits and/or invincible blocks
